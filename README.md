# Google Cloud Functions to manage Infinity One Squares

1. Call the "setInfinityOneSquare" function to create a new entity:

API: https://us-central1-niro-infinity-one.cloudfunctions.net/setInfinityOneSquare

<img src="https://github.com/InfinityPOne/niro-infinityone-squares-with-google-functions/blob/master/images/setInfinityOneSquare.JPG" alt="Infinity One logo" title="Infinity One" />

1. Call the "getInfinityOneSquare" function to read the newly created entity:

API: https://us-central1-niro-infinity-one.cloudfunctions.net/getInfinityOneSquare

<img src="https://github.com/InfinityPOne/niro-infinityone-squares-with-google-functions/blob/master/images/getInfinityOne.JPG" alt="Infinity One logo" title="Infinity One" />
